Introduction
============


Yet another social sharing button implementation for plone.

We used this at wildcard on a project and are just open sourcing it.


Features:

- Social media buttons that do not track users(call an iframe to get button, etc)
- options to choose portal types the buttons will show up on
- enable toggling buttons on every page
- options to show on top and bottom of page.
- options to only show certain buttons
- only share buttons, not +1, like, etc
